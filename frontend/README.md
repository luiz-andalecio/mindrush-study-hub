# MindRush Frontend (React)

Este frontend foi gerado inicialmente via Lovable e está sendo adaptado para integrar com um backend Spring Boot.

## Rodar em desenvolvimento

```bash
npm install
npm run dev
```

Abra: `http://localhost:5173`

## API

Por padrão, a aplicação chama a API em `/api` (proxy do Vite → `http://localhost:8080`).

Se quiser apontar para uma API externa, defina `VITE_API_URL` (ex.: `https://api.seudominio.com/api`).
